# 宅地建物取引契約の種類と効力

## 1. 媒介契約

### 1.1 媒介契約の種類

#### 単独媒介契約
- 1つの業者に対してのみ依頼する契約
- 宅地建物取引業者は、媒介契約の締結後8日以内に重要事項説明書を交付する必要がある

#### 専任媒介契約
- 専任媒介契約書には、以下の事項を記載しなければならない：
  - 媒介契約の有効期間
  - 契約の期間は3ヶ月以上6ヶ月以下
  - 契約の更新がある場合はその期間と更新回数

#### 専属専任媒介契約
- 専任媒介契約に、以下の事項が追加される：
  - 他の媒介業者との同時媒介の禁止
  - 契約の期間は3ヶ月以上6ヶ月以下

### 1.2 媒介契約の効力

#### 媒介契約成立時
- 宅地建物取引業者は、媒介契約書を作成し、交付する必要がある
- 重要事項説明書も同時に交付する

#### 媒介契約の終了
- 契約期間満了による終了
- 契約解除による終了
- 目的物の滅失による終了

## 2. 売買契約

### 2.1 売買契約書

#### 記載事項
- 目的である土地又は建物の概要
- 土地又は建物の価格及び金銭債務の金額
- 媒介の報酬の額及びその金銭債務の負担者
- 契約不履行の場合の損害の賠償の額及び違約金の日算
- 宅地又は建物の引渡しの時期
- 宅地又は建物の契約を締結する時期

### 2.2 手付金

#### 手付金の性質
- 契約不履行の場合の損害の賠償の額
- 違約金の日算

#### 手付金返還
- 売主の責めに帰すべき事由により契約が履行されない場合
- 買主は、手付金の返還を請求できる

## 3. 賃貸借媒介

### 3.1 賃貸借契約の種類
- 普通賃貸借契約
- 定期賃貸借契約

### 3.2 重要事項説明
- 敷金、礼金、保証金の取り扱い
- 管理費、修繕費の負担
- ペットの可否
- 敷地の帰属
